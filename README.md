
# SQ-Fixer

### What is it?

SQ-Fixer is a tool for fixing SonarQube issues automatically.

### Commands

##### delete

Deletes all **coded** comments.

##### var-to-let

Changes all **var** declarations in **let** declarations.

##### add-alt

Adds `alt=""` to all `<img>` and `<images>` tags that doesn't have it.
